# Assignment 4 - Third Party API

## Description
This project is a simple Movie Search Application that uses the OMDb API to fetch and display movie data such as title, year, genre, director, rating, and plot.

## API Used
OMDb API  
https://www.omdbapi.com/

## Features
- Search for any movie
- Display movie details (Title, Year, Genre, Director, Plot)
- Show IMDB rating
- Display movie poster
- Handle empty input and errors
- Loading message while fetching data

## How It Works
The user enters a movie name and clicks the button (or presses Enter).  
JavaScript sends a request to the OMDb API and displays the results dynamically on the page.
